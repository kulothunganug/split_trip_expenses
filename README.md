# Split Trip Expenses

what

a fully offline expense tracker specifically developed for managing trip expenses

why

i have been going on a few trips with friends and a lot of hackathons on different regions, we had to manage the expenses and settle the bills later, so i thought of developing an app for it, gpay offers split feature but it is restricted only to online transactions and not for offline transactions, also it does not provide a clear picture of who spent what and who owes whom

user flow

- create trip
- add members to the trip
- add expenses to the trip
- split expenses among members

features

- multiple trip
- import & export trip data
- fully offline

limitations
it is made to only be used by a single user, i.e, you can't share the trip with your friends and let them add expenses, it is only handled by the single person who's the usually the trip organizer (its me in my case) so each expense must be informed to the organizer so that they add it in the application

tech stack

- flutter
- drift for database & orm
